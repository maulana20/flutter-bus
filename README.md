# flutter_bus

## Getting Started

Pencarian jadwal bus shuttle

Langkah sebelum memulai :
- ambil paket sesuai pub yang di butuhkan
```bash
flutter packages get
```

pub yang akan di gunakan :
```bash
none
```

jalankan perintah :
```bash
flutter run
```
